# 🎮 点点够净水消消乐 (WaterGame)

一个结合教育性、趣味性和营销性的微信小游戏，通过消消乐游戏机制传播水质安全知识，展示点点够净水器的5G智能功能和品牌优势。

## 🌟 项目特色

### 🎯 核心游戏机制
- **消消乐 + 密室逃脱**：在时间限制下消除三种污染物（颗粒、微生物、化学）
- **道具系统**：PP棉炸弹、CTO激光、RO清洁波等净水器元素道具
- **关卡系统**：50个渐进式难度关卡，模拟不同水质环境

### 🚀 5G智能功能展示
- **实时监控**：模拟5G远程水质监测界面
- **智能控制**：远程开关机、强冲功能演示
- **数据分析**：TDS、余氯、PH值等水质参数展示
- **滤芯管理**：PP、CTO、RO滤芯寿命智能提醒

### 🏆 用户互动系统
- **复活奖励**：看广告复活、分享复活机制
- **成就系统**：6种成就类型提升用户粘性
- **品牌教育**：RO反渗透技术科普、成本对比分析
- **社交传播**：微信分享、好友排行榜

## 🛠️ 技术架构

### 前端技术栈
- **框架**：微信小游戏原生框架

## 🚀 快速开始

### 环境要求
- 微信开发者工具 1.05.0+
- Node.js 14.0+
- 微信小程序/小游戏开发权限

### 安装步骤
1. **克隆项目**
   ```bash
   git clone https://github.com/chenlaiyi/WaterGame.git
   cd WaterGame
   ```

2. **打开微信开发者工具**
   - 选择"小游戏"项目类型
   - 导入项目目录
   - 项目名称：点点够净水消消乐

3. **配置云开发**
   - 开通微信云开发环境
   - 修改 `app.js` 中的环境ID
   - 部署云函数 `userLogin`

4. **启动项目**
   - 点击"编译"开始运行
   - 在模拟器中体验游戏
